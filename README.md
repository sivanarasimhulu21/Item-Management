# Items Management Application (Spring Boot)

A simple Java Spring Boot backend application with a small frontend to manage items (similar to products in an e-commerce app).
This project demonstrates RESTful APIs, in-memory storage, and basic frontend integration.

## Features

->Add a new item

->Get an item by ID

In-memory data storage using ArrayList

Simple HTML + JavaScript frontend

No database required

## Technologies Used

Java 21

Spring Boot

Spring Web

Maven

HTML, JavaScript

Embedded Tomcat Server
